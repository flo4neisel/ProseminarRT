Allgemeines:
Git ist eine hilfreiche Software zur Versionsverwaltung eurer Arbeit (gut geeignet für Code z.B. Latex oder Matlab).
Es ist keine Pflicht, Git zu benutzen, aber ggf. hilfreich.


Das Wiki-Book:
zum Einlesen in Git, wichtige Befehle wie commit, stash, pull, push, fetch, merge, branch, etc...
Da gibt es sicherlich auch gute Videos bei Youtube, etc...


Git:
die eigentliche Git-Software, die auf eurem PC installiert sein muss, um die git-Funktionalität zu nutzen.
Ist diese installiert, könnt ihr im Explorer eine git-bash öffnen, über die ihr git-Befehle eingeben könnt.
Es gibt dann auch eine git-gui, die auch für einfache Befehle wie zB commit und push genutzt werden kann.
--> https://git-scm.com/downloads


Turtoise git:
Zur Nutzung von git nicht notwendig (benötigt aber die Installation der reinen Git-Software),
stellt aber eine Benutzeroberfläche bereit, die die Arbeit stark vereinfacht.
Zudem sind Befehle wie pull, fetch, etc. dann auch im Explorer direkt aufrufbar (Rechtsklick --> TurtoiseGit --> ...).
--> https://tortoisegit.org/
